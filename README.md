# Fence Assignment

Deployed Link - https://fence-assignment.netlify.app/

### Try on your Machine

#### Clone
```bash
git clone https://github.com/OGogte/swipe-assignment-frontend.git
```

#### Navigate to the Project directory
```bash
cd swipe-assignment-frontend
```

#### Install Dependencies
```bash
npm i
```

#### Run Project
```bash
npm run dev
```
